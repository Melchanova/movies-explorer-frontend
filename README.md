# movies-explorer-frontend

Дипломная работа по созданию интерактивной страницы. Пользователь может искать фильмы по ключевым словам и добавлять их в избранное в личном кабинете.

## Основные инструменты:

- HNML, CSS;
- Java Script;
- React;
- Адаптивная верстка
- Flexbox
